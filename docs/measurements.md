## Measurements Module (Frontend)

### Pages

- List: `/measurements`
- Create: `/measurements/new`
- Detail/Edit: `/measurements/:id`

### Edit rule

- Only one Edit button is used (top header).

### APIs used

- `GET /api/measurements`
- `GET /api/measurements/{id}`
- `POST /api/measurements`
- `PUT /api/measurements/{id}`
- `GET /api/measurement-fields?garment_type=Suit|Shirt|Pants`
- `GET /api/staff`
