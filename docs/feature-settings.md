## Feature: Settings (Frontend Plan)

### Sections

- Profile/Shop: name, address, phone, logo
- Users & Roles: list users, create user, assign role
- Notifications: OTP + appointment reminders templates/toggles
- Upload Settings: image types, max size, crop defaults
- Security: token expiry/password policy (future)

### Routing (suggested)

- `/settings`
- Tabs inside settings:
  - General
  - Users
  - Notifications
  - Uploads
  - Security
